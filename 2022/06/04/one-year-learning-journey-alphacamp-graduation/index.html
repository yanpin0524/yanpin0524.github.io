將近一年的學習之旅，我從 AlphaCamp 畢業了～
2022 June 4th
網頁開發
心得分享

我這將近一年的 AlphaCamp 學習之旅終於要告一段落了，現在回想起之前的辛苦，彷彿一切都值得了。

我是一個高中生，對 AI 、資料分析 很感興趣，所以我開始自學 Python 直到我在學習如何爬蟲時，我突然意識到我不知道要如何把作品給不了解程式設計了人看，然後當時的我也很單純的以為，只要會寫網頁就能夠把作品呈現給其他人看了，所以我就報名了 AlphaCamp 的課程，好笑的是，現在我的 JavaScript 比 Python 還厲害(我的 Python 還只會爬蟲而已)。

學期一時我還自大狂妄的以為我學過 Python 所以我一定比較厲害(就是個小屁孩)，直到第二學期開始我開始有點跟不上進度了，心態也開始從原本的我最厲害，慢慢轉變成我也只是個普通人，也開始觀摩起別人的作業，發現原來還有這種解決方式啊，並打從心底覺得其他人好厲害。

然後到了第三學期我徹底被壓垮，學習內容變得更艱難，學習量也變得更多，作業開始趕到最後一刻才交，甚至是遲交，我徹底崩潰了，我開始想『其他人都已經開始預習下週的進度了，我呢？甚至連這週的都還沒上完，而且其他人還是一邊工作一邊學習的』我開始認為或許我不適合這條道路，但好險有我的家人一直鼓勵我，跟助教的期中評價讓我重拾了信心，而且我的作業第一次被同學留言了他問了一些問題並說我好厲害，雖然他可能不知道，但那一句稱讚對那時的我意義重大，之後那位同學邀請我一起進行 Twitter專案，我合作的很開心，感謝 Yun Hung助教與那位同學陪我走完了這艱難的第三學期，甚至可以說我能夠從 AlphaCamp 畢業都是多虧了你們，謝謝。
